Question-1: What are the different ways to select an element in the DOM?
I can select DOM elements using getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. Other options include document.forms, document.images, and document.links for specific elements.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML shows HTML with tags, innerText shows only visible text, and textContent shows all text, including hidden ones, but ignores HTML tags.
Question-3: What is event delegation in the DOM?
Event delegation lets a parent element handle events for its children using event bubbling. Instead of adding many listeners, one listener on the parent detects clicks on child elements efficiently.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event starts at the target element and then "bubbles" up to its parent elements. This means the event triggers on child first, then on the parent.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element, use document.createElement(). Add it with appendChild(), and remove it with removeChild().
